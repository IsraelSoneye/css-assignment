Second Section: What I have learnt about css

CSS is a very important tool in web development. It is used to style html content and makes the web page look nice to users. CSS can be use to set text size and color, create web page layout, add padding, border, margin etc. Using css, one can select id, element, class and different other properties for specific design.

Third Section: Difference between Padding, Margin and Border

Padding: Padding is the space around the text frame. It is the gap between the text area and the border.
Margin:It is the space outside the border of a web page.
Border: It is the frame around the text that demacates between the padding and the margin.
